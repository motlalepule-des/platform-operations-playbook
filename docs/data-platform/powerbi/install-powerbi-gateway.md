# Install Power BI Gateway

## Purpose
Install the on-premises data gateway to connect Power BI Service to local data sources.

## Prerequisites
- Windows machine that is always on
- Power BI account
- Internet access
- Access to local SQL Server

## Steps
1. Download **On-premises data gateway**
2. Run the installer
3. Sign in with your Power BI account
4. Register the gateway
5. Give the gateway a clear name
6. Set a recovery key

## Recommended Host
Install the gateway on:
- A stable server
- A machine with reliable connectivity to SQL Server
- A machine not frequently powered off

## Next Step
Configure the SQL Server data source in Power BI Service.
