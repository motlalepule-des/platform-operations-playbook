
# Use Case: Publish Local API for External Testing

## Scenario

A developer needs to expose a locally running API so external systems can test integration.

Example:

Local ASP.NET API running on port 5000.

## Solution

Use a tunneling tool such as:

- ngrok
- Dev Tunnels
